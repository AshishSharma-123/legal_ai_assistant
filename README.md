Legal AI Assistant
---
A Flask-based AI Legal Assistant web application that allows users to sign up, log in, and ask legal-related questions using Google's Gemini AI model.

The project also includes a secure authentication system using Xampp Server - MySQL and password hashing.

---
🚀 Features

User Signup & Login system

Password hashing using Bcrypt

MySQL database integration

AI-powered legal assistant

Clean Flask web interface

Environment variable security

Gemini AI integration for answering legal queries
